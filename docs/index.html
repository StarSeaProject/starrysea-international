星之海 海外版API

This is starrysea International Ver. API Documentation

API主机地址: api.starrysea.org

封面静态文件地址: api.starrysea.org/static/img/activity/

PDF静态文件地址: api.starrysea.org/static/pdf/

创建一个新的管理员

创建一个新的管理员🆕 需要管理员权限

属性 介绍
username (必填) 用户名
password (必填) 密码
POST
/users/create

Example response

{
    "success": true,
    "response": {
        "_id": "{对象ID}"
    }
}

创建一个新的活动

创建一个新的活动🆕 需要管理员权限

此请求包含文件上传，请妥善处理

属性 介绍
name (必填) 活动名称
content (必填) 活动介绍
summary (必填) 活动简介
endTime (必填) 结束时间
cover (必填( 文件 )) 封面文件
POST
/activity

Example response

{
  "success": true,
  "response": {
    "date": {
      "created": "{创建时间戳}",
      "endTime": "{结束时间时间戳}"
    },
    "_id": "{对象ID}",
    "name": "{名称}",
    "content": "{内容}",
    "summary": "{简介}",
    "cover": "{封面文件}",
    "__v": 0
  }
}

获取活动列表

按页面数和限制获取活动列表

GET
/activity?page={页面数}&limit={限制数量}

Example response

{
    "success": true,
    "response": [
        {
            "_id": "{对象ID}",
            "name": "{活动名称}",
            "summary": "{活动简介}",
            "cover": "{封面图片}"
        }
    ]
}

创建一个新的作品

创建一个新的作品🆕 需要管理员权限

此请求包含大量文件上传，请妥善处理

属性 介绍
name (必填) 活动名称
content (必填) 活动介绍
summary (必填) 活动简介
endTime (必填) 结束时间
cover (必填( 文件 )) 封面文件
pdf (必填( 文件 )) PDF文件
images (必填( 文件 )) 图片文件,最多32张
POST
/work

Example response

{
  "success": true,
  "response": {
    "file": {
      "images": [
        {
          "filename": "{图片文件}"
        }
      ],
      "cover": "{封面图片}",
      "pdf": "{PDF文档}"
    },
    "created": "{创建时间戳}",
    "_id": "{对象ID}",
    "name": "{名称}",
    "summary": "{简介}",
    "view": 0,
    "__v": 0
  }
}
